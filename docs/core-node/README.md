# Core-нода

> **Статус:** Этап 2 (планируется)

Core-нода — это скрытый узел ядра сети, который маршрутизирует трафик от Entry-нод в интернет.

## Роль в архитектуре

```
Клиент → Entry-нода → Core-нода → Интернет
                      ^^^^^^^^^^^
                      Вы здесь
```

## Характеристики

| Параметр | Значение |
|----------|----------|
| Функция | Маршрутизация трафика в интернет |
| Жизненный цикл | Долгоживущая, защищённая |
| Количество | Относительно небольшое |
| Связность | Полная связность между Core-нодами |
| PKI | Промежуточный CA, выпускает сертификаты для Entry-нод |

## Задачи Core-ноды

1. Приём трафика от Entry-нод
2. Маршрутизация во внешний интернет
3. Управление PKI (выпуск сертификатов для Entry-нод)
4. Хранение списков Entry-нод
5. Мониторинг и health-check Entry-нод
