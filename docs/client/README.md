# Клиентские приложения

Этот раздел содержит информацию о клиентских приложениях для подключения к сети.

## Поддерживаемые платформы

| Платформа | Приложения |
|-----------|------------|
| Windows | v2rayN, NekoRay |
| macOS | V2RayXS, NekoRay |
| Linux | NekoRay, v2rayA, Xray CLI |
| Android | v2rayNG, NekoBox |
| iOS | Shadowrocket, Stash |

## Параметры подключения

Для подключения необходимы:

| Параметр | Описание |
|----------|----------|
| UUID | Идентификатор пользователя |
| Адрес | Домен Entry-ноды |
| Порт | 443 |
| Транспорт | gRPC |
| Путь | Секретный путь |

## Получение конфигурации

Конфигурацию можно получить:

1. **VLESS-ссылка** — импортируется одним кликом
2. **QR-код** — сканируется мобильным приложением
3. **Ручная настройка** — вводится вручную в приложении

---

*За конфигурацией обращайтесь к администратору.*
